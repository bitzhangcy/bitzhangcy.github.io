<center><span style="font-family: STKaiti; font-size: 35px; color: #3BB9FF; font-weight: bold;">张弛从来道似弓，聊将数语示儿童<br>
悠悠二百余年事，都在衰翁一梦中</span></center>

## <font face = STKaiti >三千年未有至大变局</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;晚清史是我最早感兴趣的一段历史，从曾国藩到李鸿章再到袁世凯，这是一段波澜壮阔的历史图卷。处于巅峰的皇权遇到了三千年未遇之大变局，无数的大儒皓首穷经，可惜祖先和历史都没给我们一个答案，我们只能自己去寻找答案。天朝的朝贡体系和西方侵略者爆发了严重的冲突，我们对国际关系处理被资本主义重新定义了。中国人遇到了挑战，神州大地也曾沉沦过，但是中国人也最终奋起了，打败了以国运相争的日本 人，重新主导了亚洲的局势。</font>**

## <font face = STKaiti >近代史开端</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;关于中国近代史的开端，这么多年学界一直没有一个统一的定义，中国大陆这边的官方定义是第一次鸦片战争，徐中约先生在《中国近代史》一书中认为是1600年，大致可以认为是明清之际。我本人倾向于徐中约先生的观点，隆庆大开海无形中促进了东西方的交流。需要指出的是，东南的对外贸易甚至还促进了资本主义的出现，但是这是在特定的历史背景下，就是士绅集团联合与皇权集团进行对抗，甚至还闹出了苏州暴动对抗皇帝缇骑的事情。历朝历代，闻所未闻，不过这资本主义的曙光只是一闪而过，历史的惯性使得帝国仍然走向了封关闭国的天朝上国状态。</font>**

## <font face = STKaiti >天朝礼法</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;清史我是大致分为康乾盛世、嘉道咸乱世初现和晚清风云这三个部分。晚清的故事并不是一蹴而就的，帝国沉珂难返，又逢三千年未遇之大变局。四书五经，圣人之言都没有告诉帝国的掌舵者如何去面对这乱局。天朝朝贡体系是中华帝国融入国际体系中的最大障碍，当是时，国际体系初成，中华帝国仍在为外交礼仪为苦恼。第一次鸦片战争之后，为了解决英法使节进京换约的问题，旧式官僚们想尽了一切方法，不惜牺牲了中国的关税自主权都要拒绝英法使节进京。从后世的角度来看，我们是错失了很多机会去换取一些礼义上的所谓大防，但是在当时的人看来，天朝大国的脸面是维护这个朝贡体系和维持大一统王朝的利器。所以读晚清史一定要站在当时的时代背景上进行解读，事后诸葛亮的方式读晚清留下的只有疑惑，湘淮两系都是想有所作为的英雄，奈何处处有人掣肘，这就是中国人长此以往的内卷特性，兄弟阋于墙，防家贼甚于防外贼。</font>**

## <font face = STKaiti >晚清大舞台</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;晚清史绕不过的几个人，<span style="color: #EF1A08;">咸丰皇帝，慈禧太后，曾国藩，李鸿章，醇亲王，恭亲王，庆亲王，左宗棠，张之洞，翁同龢，袁世凯和摄政王载沣这几人</span>，当然了还有湘淮两系的很多人，还有庚子国变之后崛起的一批人，更重要的是还有一批满人亲贵和重臣。当然了，在晚清这个舞台上唱戏的首先是咸丰皇帝，然后就是慈禧太后，最后是摄政王载沣，这三个人里面，慈禧太后主掌朝政的时间最长，可以说是挽大厦之将倾，救中国维护了大一统，将平衡的艺术发挥到了极致，玩弄天下英雄与鼓掌之中，四十年来功过参半。咸丰皇帝是一代治皇，可惜生不逢时，大清已是帝国黄昏，没有张居正那样的能臣下一剂猛药是难以中兴的，最后西狩承德避暑山庄，含恨于此。这几十年中华帝国也经历了三任皇帝，同治，光绪和宣统，其实同治皇帝还是有点存在感的，在他手上毕竟还有个同治中兴，依靠湘军功勋集团平定了中国历史上声势最浩大的农民起义运动，开启了一个自强的时代洋务运动，也平息了第二次鸦片战争，中华帝国再次出现了中兴的局面，可惜盛世的幻境很快破灭。同治皇帝的死亡也是历史的迷案，穆总皇帝死于十九岁，就如同帝国末年一样，小皇帝是帝国的标配。穆总无子嗣而崩，慈禧从醇亲王出抱养载湉继祧穆总皇帝宗庙。中国的宗法继承制度是非常有意思的，以后可以开一期详细谈一谈大宗和小宗之间的关系。光绪皇帝算是唐昭宗一类，本人没有祖辈的才智，却又想不断折腾，最后输光了所有的老本，徒留下一声叹息。光绪皇帝有很好的本钱，同治中兴的余光尚未完全褪去，慈禧太后在幕后平衡着地方与中央的脆弱关系，可惜暗弱无能，竟想依靠康有为的君主立宪势力完成革新。光绪皇帝一生的亮点就是百日维新，可惜连地方的汉人督抚大员都不买账，唯一可圈可点的就是建立了京师大学堂，也算是开了中国革新之先。甲午战败，忍辱负重才是历史的主流，这位皇帝太心急了，可惜没有他的祖先康熙皇帝的才谋和机会。</font>**

<center><span style="font-family: STKaiti; font-size: 25px; color: #3BB9FF; font-weight: bold;font-style: italic">玉宇将倾人祸生，少年扶危却未成<br>
瀛台忽落潇潇雨，最是人君忆倾城</span></center>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;宣统皇帝算是安乐公之后过得最好的一位末代皇帝，他一生做过三次皇帝，第一次其实是父亲摄政王载沣代行皇帝事，第二次是张勋复辟，本质上就是一次闹剧，第三次是伪满洲国，不过是傀儡皇帝。可以这么说，宣统皇帝的一生就是吉祥物的一生，用皇帝的身份在全中国招摇撞骗。说一千道一万，晚清这个舞台主要就是慈禧太后主导的，她和现在的农村老太太没什么区别，那篇错字满篇的两宫懿旨就完成了辛酉政变，不失为一代政治强人。</font>**

## <font face = STKaiti >负重前行的帝国</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;言归正传，从第一次鸦片战争和第二代鸦片战争的失败不是单纯的军事失败，而是帝国的全面崩塌，吏治腐败已经无可挽回了，战争史上最奇葩的一幕也出现了，中英两国交战，中国老百姓在边上观战，甚至还和英国人做生意，弄出了三元里大捷这样的闹剧。就像英国使节马嘎尔尼所说，<span style="color: #EF1A08;">大清帝国只是一艘破烂不堪的旧船，只是幸运地有了几位谨慎的船长才使它在近150年期间没有沉没</span>。<span style="color: #063CE7;">不得不承认，嘉庆皇帝从他的父亲手上接过的就是一个千苍百孔的帝国，他们祖孙三代不停地努力，可这艘大船已经破败不堪了，再勤奋的船长也不能避免沉没的命运</span>。咸丰皇帝是一个奋进的皇帝，可惜这西风东渐的局面再也在二十四史和四书五经中找到答案，四夷皆是临御天下，可惜西方人只想通商，但是这种行为挑战了中国的朝贡体系，鱼与熊掌不可兼得，咸丰皇帝选择维护旧有的秩序。咸丰皇帝也开了皇帝西狩的传统，两次英法战争的阴影笼罩在咸丰皇帝的生命中，他也锐意进取，重用汉臣，试图整肃吏治，奈何天不假年，最后含恨崩逝。可能是清德式微，一代不如一代，到咸丰皇帝这只有同治皇帝这一个继承人，想不到咸丰皇帝是清朝最后一个没有绝嗣的皇帝。两次鸦片战争，并不是旧式军队没有血性，并不是武器落后，只不过自康熙朝开始，天下就武备松弛，八旗不行了就组建绿营，绿营腐化就让地方组建团练，团练不行就让地方大员配合地方士绅编练私军。这样军队的战斗力是上来了，明末兵为将有的乱象又再次重现。这些地方士绅这么积极地配合编练私军可不是为了保卫自己的土地财产，而是为了掌握地方的实权，这就为地方做大提供了得天独厚的土壤。</font>**

## <font face = STKaiti >鸦片战争</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;谈起咸丰皇帝，就不得不提到虎门销烟，当时的中国人已经普遍意识到了鸦片的危害性。林则徐也清醒地认识到“此祸不除，十年之后，不惟无可筹之饷，且无可用之兵”，开始了浩浩荡荡的虎门销烟。林则徐也算是清末开眼见世界的第一人，与魏源合著《海国图志》，他也深感中国吏治疲敝，中国之败不在于武器落后而是民心离散所致，但是他后来被发配新疆又获启用，也不得不接受官场的陈规陋习，北京城是一座雄城，历代非不到亡国的地步不会失北京城。第二次鸦片战争仍然是贸易战，希望北京朝廷能够开放更多的通商口岸。但是隆庆大开海贻害无穷，造成东南动荡，所以清朝统治者清醒地看出了这一点，不希望开放过多的口岸造成中国人心离散。但是大势所趋，时代的大潮已经冲开了中国的大门，所以咸丰皇帝也只能无奈地割地求和。咸丰皇帝执政最后的一段时间，国家已经非常危险了，内有太平天国起义，外有英法侵略者的咄咄逼人，奈何中华帝国已经迎来了末日的先兆。咸丰皇帝也算是鞠躬尽瘁死而后已，最后病死在承德避暑山庄，被国事艰危所压垮。</font>**

## <font face = STKaiti >辛酉政变</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;咸丰皇帝死之前还是埋了炸弹，有清一朝的权力交接都伴随着波云诡谲，咸丰死之前任命了顾命八大臣共辅朝政，这八人分别是:载垣、端华、景寿、肃顺、穆荫、匡源、杜翰、焦佑瀛。肃顺还是比较有本事的，但是这里第一个大雷出现了，咸丰还给两宫皇太后慈安和慈禧分别留了“御赏”和“同道堂”的两枚御印，八大臣所拟诏书必须盖上这两枚御印才能生效，相当于皇家还是要保留监督权。但是慈禧和肃顺两个人都想独揽权利，这样矛盾就出现了，恭亲王也被咸丰摆了一道，同治皇帝的亲叔叔竟然没有授予实权。这样恭亲王就和两宫皇太后形成了统一战线，通过配合抢先回到了北京，发动了政变，史称辛酉政变。政变的过程当然是尔虞我诈，充满了血腥，但是这一次政变深刻地改变了清朝历史的走向，八大臣的犹豫和不果决造就了他们的悲剧。这次政变的直接影响就是慈禧太后开始主导清朝走向，历史又一次走向了未知的深渊。清朝对后宫的管理特别严格，后宫干政也就是孝庄皇太后以皇帝年幼摄政，自此以后，后宫不得干政已经成为了铁律。更有趣的是，慈禧本人死前也交代，后世不允许再有女人主政。下面就开启了同治时代了。</font>**

## <font face = STKaiti >两宫同治</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;同治这个年号很有意思，取自“两宫同治”之意，同治一朝还算国泰民安。湘军平定了太平天国，而后曾国藩又自解湘军，</font>**
<center><span style="font-family: STKaiti; font-size: 25px; color: #3BB9FF; font-weight: bold;font-style: italic">倚天照海花无数，流水高山心自知</span></center>
**<font face = STKaiti color = Black size = 4>但是湘军系已经控制了富庶的东南，清末四大疑案之一的刺马案就紧接着发生了。这个案子所有人都知道是湘军系所为，但是没有人敢把这件事拿到明面上来说，这么做无异于逼反湘军，中国已经经历了一次太平天国的浩劫，再也不能承受如此大劫了。 很多人可能会疑问为什么当年曾国藩没有造反，这是中国儒家面临的传统难题。曾国藩和湘军系将领之间是师生关系，与左宗棠等人是朋友关系，也就是说，很难在传统儒家的意义上将这种关系转化为君臣关系。还有一个很可怕的问题，就是如果东南一旦大乱，天下就会如五代十国一样陷入长久的动乱。自从中国经济中心南移，将首都定在北京，漕运就关乎天下命脉，唐朝的时候西都长安就已经缺粮了，皇帝不得不带文武百官东去洛阳就食。唐朝就是依靠着忠心耿耿的东南节度使们才能续命一百多年，这里漕运和海运之争又是一个值得讨论的话题，百万漕工衣食所赖？</font>**

## <font face = STKaiti >团练运动</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;同治一朝的开始，便是如何平定太平天国，天平天国运动很难去评价其的正义性，但是毫无疑问的是农民起义的局限性，北伐军甚至一度打到了天津，可惜最终功败垂成。为了平定太平天国，咸丰皇帝甚至下诏：<span style="color: #EF1A08;">无论何人，凡攻克敌巢南京者，必封王爵</span>。自秦皇推行大一统以来，以异姓封王者必为篡位逆臣，可以看出太平天国运动已经把清朝最高统治者已经被逼急了，上一次还是李定国两蹶名王的时候，顺治皇帝动了划江而治的念头。湘军配合淮军在基本很快肃清了地方上的太平天国了，就只剩下攻下南京城了。这里顺便说一句，打下南京城拜了明太祖的孝陵，他就具有了法理上的合理性，这也就是为什么南京是一座重镇。从明朝开始，南京真正被攻破只有一次，那就是1937年日本人进攻南京顺利打下了南京，其余时候只不过是像明末是主动投降或者围城至粮尽。这一次曾国藩和曾国荃两兄弟就是有一次发挥了结硬寨，打呆仗的风格，把南京城团团围住，最后洪秀全被围至粮尽而天平天国灭亡。这里提一嘴，南京明城墙有十三个门，是绝无仅有的东南重镇。南京城破，据说曾家两兄弟私吞了太平天国积累的巨量财富，这里当年的慈禧也曾问过曾国藩，曾国藩的满分回答，“臣也觉得此事甚是奇怪”。曾家两兄弟平定了太平天国，可谓是有再造大清的功劳，但是慈禧和恭亲王商量再三还是决定不能封一个汉人王，三藩之乱可谓是前车之鉴。最后曾国藩封一等毅勇侯，死后谥号“文正”，可谓是有清一朝汉人所受的最高荣誉。这里还有件小事，后来要报销湘军军费的时候，恭亲王一看这一定是一笔烂账了，要是真算清楚了那就是无形之中逼反了湘军，就一笔勾销了，真所谓是水至清则无鱼，人至察则无徒。</font>**

## <font face = STKaiti >封疆大吏</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;湘军和淮军伴随着太平天国的余晖崛起，也从此开启了他们主导东南地方大权的时代，这里不得不说到一个概念，就是晚清中兴四大名臣，有两种说法，说法一：曾国藩、左宗棠、李鸿章、张之洞；说法二：曾国藩、左宗棠、胡林翼、彭玉麟。这里胡林翼和彭玉麟大家可能不太熟悉，他们都是典型的湘军系，主导了地方的洋务运动。随后，中央的有识之士和地方的士绅集团就开始合作开启了洋务运动，开启了一个自强时代，洋务派积极开办了江南制造局、福州船政局、安庆内军械所等近代军事工业，开办轮船招商局（后世赫赫有名的招商局集团）、开平矿务局、天津电报局、唐山胥各庄铁路、上海机器织布局等民用企业。洋务派的观点也很有意思，<span style="color: #9A11DA;">“为以夷攻夷而作，为以夷款夷而作，为师夷长技以制夷而作”，“中学为体，西学为用”</span>，这也就表明了此时的有识之士并没有意识到是中华帝国的体制出现了问题，而是认为器不利而已，这也就陷入了明亡的陷阱中，为了平辽东而加饷，加饷又导致农民起义，政府财政状况进一步恶化。不能否定洋务运动的历史价值，这是中华帝国对西方坚船利炮的第一次反抗，但是和王安石变法一样，由于大部分儒家臣子的不合作，洋务派不得不和奸猾贪腐之辈合作，最后导致了洋务运动没有发挥出预想的作用。恭亲王主导的中枢给予了洋务派们最大的支持，但是中央和地方上都存在强大的阻力，洋务运动的成果也主要落在了东南各省上。同治中兴是这个古老帝国的最后的曙光了，这也是封建王朝最后一次中兴了，虽然这和当时主政的同治皇帝关系不大，但是叔嫂二人共同辅政也是有可圈可点之处。</font>**

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;洋务运动是慈禧影响中国历史走向的开始，湘军系、淮军系和满人文物亲贵都被她一次次地平衡了。天津教案使得曾国藩从神坛落下，也意味着曾家对朝政的影响力开始下降，李鸿章接过他的老师的旗帜，担任八大总督之首的直隶总督，开始了他以知洋务的下半生。甲申易枢之前，李鸿章在推进洋务的路上还算顺利，包括恭亲王和文祥在内的满人能臣都积极支持他，这一点在北洋水师的建立上是有目共睹的。太平天国被平定之后的十年算是清朝的黄金十年，这十年也算是治世，旧有的秩序伴随着先进的技术在重建。日本也因为黑船事件开启了倒幕运动，天皇开始重新掌权，并制定了称霸亚洲的目标，所以他们的第一步是征服朝鲜，第二步以朝鲜为跳板政府中国。但是他们又觉得台湾很有潜力，所以于1874年悍然侵犯台湾，当时的清政府还不是纸老虎，一面军事反击，一面派沈葆桢去谈判最终逼迫日本侵略者退出台湾，恭亲王的这一手操作还是很厉害的。甲申易枢之前，清政府的大部分操作都还是能看懂的，可惜从鬼子六恭亲王退出朝政以后，清朝政府的手段也愈发捉襟见肘，大部分时候只能依靠地方督抚大臣和慈禧的私交勉励维持国家不乱。日本妄图侵占台湾也让当时一部分实干的有识之士感到了压迫感，自曾国藩起兵平乱以来，经世致用已经成为了很多有识之士的警世格言，蕞尔小邦都幻想侵占台湾宝岛，这一件事间接也促进了台湾独立建省，这里就要提到台湾的首任巡抚刘铭传，此人也是经世致用之才，奈何时不我待。</font>**

## <font face = STKaiti >宗庙统序</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;1875年就来了，大清水德更加微薄，这一年同治皇帝崩，咸丰皇帝绝嗣，这里又开始了皇权的博弈，最终选定了醇亲王之子载湉继祧同治皇帝，改元光绪。这里清朝这条行将沉没的巨舰又走向了不可见的历史迷雾之中，如果同治皇帝不死能够及时亲征，那么未来的皇帝和慈禧太后就是母子情深，也许不会发生那么多事情，皇帝能够顺利亲 征，在洋务派的大臣的辅佐下说不定也能成为一名治世皇帝。可惜历史从来没有假设，中华帝国最后的希望已经破灭。光绪皇帝是幸运的，他以幼年继位，有恭亲王为首的军机处尽力左支右绌，有洋务派在地方进行近代化努力，有慈禧太后在中央运筹帷幄，留给他成年的政治遗产是相当丰富的，可能是幼年进宫缺失父母亲的疼爱，他的性格是有严重缺陷的，他选择相信他的老师翁同龢，一个只知君君臣臣父父子子中的腐儒与旧式官僚。光绪皇帝继位的前十年还是四海承平，可惜接下来的十年大清已经是风雨飘摇了。</font>**

## <font face = STKaiti >塞防与西北大捷</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;西方侵略者自东南沿海而来，这些都是帝国的癣疥之疾，只需要用一些贸易权和尚未明晰的法外治权来进行交换。乾隆皇帝引以为傲的就是一劳永逸地解决了准葛尔部，一举巩固了西北赛防，承平不过百年，西北重地又生肘腋之变。但是同时，以李鸿章为首的海防派又联合起来，要求舍弃塞防专注于海防，毕竟西方侵略者都是从海路而来，组建一支强大的海防力量也是当务之急。但是当时的清廷财务状况已经不容乐观了，只能选择支持海防或者塞防，但是这两派的力量又是势均力敌，最后考虑“克复新疆，所以保蒙古；守卫蒙古，所以保京师”，清廷决定支持左宗棠收复新疆和伊犁，当时伊犁是为俄国人所占，是曾家曾纪泽谈判才得以收复。左宗棠不愧为中兴名臣，坐镇后方运筹帷幄，很快就收复了新疆，但是由于海防派还占据了政局的一大部分，所以收复新疆的过程中也是处处掣肘，但是红顶商人胡雪岩帮助左宗棠走出了财政泥沼之中。大明王朝有一句特别有意思，那就是，<span style="color: #F50F0B;">“上下挥霍无度便掠之于民，民变在即便掠之于商”</span>。胡雪岩就是狡兔死，走狗烹；飞鸟净，良弓藏，最后只落个破产的下场。从现代中国的角度来看，收复新疆意义重大，至少为中国的西北部留下了重要的屏藩，中国海防没那么重要，因为一支脱胎于旧式军队的海军，即使中高级官员曾经在英国进修过，也是难以摆脱旧式军队的习气。左宗棠在清末的历史地位是无可置疑的，但是他一生有一个遗憾，那就是没有中进士，甚至在平定新疆的过程要回京参加会试，最后不得不以慈禧太后授予他一个同进士出身。当洋务运动开展得轰轰烈烈的时候，传统还是难以破除，左宗棠还是痴迷于进士出身，甚至不惜耽误新疆战事。当然了，那句评价<span style="color: #F50F0B;">“国家不可一日无湖南，湖南不可一日无宗棠也”</span>，这句话一点也不为过。当然了，左宗棠也是慈禧太后牵制湘军系统的利器，当年曾国藩不肯造反的原因除了身前身后名，还有顾虑左宗棠的大才。当然了，左宗棠还曾平定西北西北回民之乱，比之凌烟阁二十四功臣的战神李勣，今日国家西北安宁不能忘了文襄公当年的呕心沥血。</font>**

## <font face = STKaiti >刺马案</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;自朱元璋废宰相以来，明清两朝再无宰相之名，但是宰相之实还是不能避免的。明朝我们一般认为内阁首辅就是宰相，清朝的就比较复杂，我们一般认为首席军机大臣加四殿两阁大学衔的就被称为宰相。当年翁同龢为了得到大学士头衔可谓是不择手段，最后想不到只落得交家乡官员严加看管的下场，两代帝师，度量之小，历朝未见。明清两朝虽然皇权达到了巅峰，但是东南豪族的势力还是非常大。明武宗，明光宗和明熹宗这三位皇帝的死亡都很蹊跷，其中明武宗和明熹宗这两位不遵循常理的皇帝都是落水惊悸而死，为他们治病的太医院医生最后都不知所踪，最少可以说这两位皇帝的驾崩肯定有南直隶势力的黑手，明末梃击案、红丸案都和明光宗相关，可以看出来东南豪族对皇宫内卫的腐蚀程度了。满人皇帝虽然吸取了明朝的教训，加强了皇帝自身的安全保证，但是到了晚清的时候，中央政府任命的两江总督马新怡竟然在赴任之前遇刺，此即此地无银三百两，清末的刺马案就由此产生，中央派去的刑部侍郎竟然不敢查此案，直接辞官回家躲灾去了。其实也可以看出，清末的东南势力已经铁板一块了，不是清初的时候雍正皇帝推行官绅一体纳粮时候那般听话了。当然了，老到的慈禧太后后来又利用了清末的另一大案“小白菜案”沉重了给了以左宗棠为首的湘军系中的“两湖派”一击，也算是报了当年的一箭之仇，太平天国刚刚平定，中央手里几乎没有任何像样的军事实力，也只能捏着鼻子认了这对中央政权的挑战。往深处说，刺马案本身也和太平天国运动有关，曾氏两兄弟攻破了天京，竟然对外宣称太平天国国库是空的，这让寄希望于这笔财富来填补财政窟窿的清政府心有不甘，但是此时又不敢逼湘军太紧，万一真逼反了这些人，满人连回黑水白山之间的可能性都没有，这个时候就看出了恭亲王为首的中枢是多么清醒。但是慈禧太后还是不死心，还是派了心腹马新怡去查这件案子，最后马新怡死得不明不白，一帮人揣着明白装糊涂。太平天国国库我相信是有大量财富的，但是城破之后被乱军哄抢一空了，曾氏兄弟也分到了一大笔，他们也不敢找手下人要，湘军的特殊体系也就是层层效忠的体系这个时候起到了反作用。</font>**

## <font face = STKaiti >海防</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;左宗棠收复新疆之后，到甲申易枢这段时间，清朝中枢还是在正常运作的，恭亲王配合慈安太后还是能压制住慈禧太后个人的穷奢极欲的。但是1884年慈安太后被毒死之后，恭亲王也就从此下课不再过问国事了。以李鸿章和恭亲王为首的洋务开明派不断开拓进取，朝中的顽固派此时也跳出来兴风作浪，历朝历代都有的清流派又开始了怼天怼地的节奏，自身正直不一定代表是大公无私的，清流往往是结党营私，不论是非，只论成败。我自己也曾思考过这个问题，朝堂之上不能没有清流，但是结党营私的清流是万万不能有的，世人都知道严嵩父子是奸臣，但是他的继任者清流徐阶就是好人吗？上海的徐家汇是怎么得名的，不就是那片地都全是徐阶家的，七千万亩良田都是一家一姓的，要不是张居正这个有明一朝唯一的名相出来收拾烂摊子，北边用戚继光对抗鞑靼，南边丈量天下田亩，用银两代替实物纳税，一条鞭法给大明续命了几十年。清流并不一定是好人，奸臣不一定全是霍乱朝政的人，忠奸难辨才是朝堂上的常态。被慈禧太后压了几十年的袁世凯，等慈禧太后一死便立即亲手将清帝国葬送。但是清末的清流对国家是无益的，处处掣肘那些想干实事的人。慈安太后死后，恭亲王的权势就一日不一日了，甲申易枢并不是拍脑袋决定，而是已经谋划了好久的一件事，恭亲王的摄政权利首先被剥夺，然后中枢被一次次打击，最后不得不隐退回家。北洋水师和恭亲王有着不解之缘，当左宗棠解决了西北的塞防问题，清廷的财政困局得到了缓解，李鸿章就在恭亲王的配合下组建了北洋水师，当然当时也有南洋水师，是当时南洋大臣沈葆桢主动提出将南洋水师的经费让给北洋水师，全力发展北洋水师。1888年北洋水师正式成军，但是其大部分经费都是恭亲王主政期间的划拨给北洋水师的，可以说，如果恭亲王一直主政中枢，那么甲午之战北洋水师未必没有一战之力，让日本这个蕞尔小邦第一次赌上国运就失败了，也就再也没有这个东亚小霸了。</font>**

## <font face = STKaiti >甲申易枢</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;中法战争算是晚清史的一个转折点，从这个时间点，晚清政府彻底进入了亡国倒计时，慈禧太后埋得雷太大了，这些炸弹不是那么稚嫩的满人亲贵能解决的，就像西晋接手了五胡这个隐患最终在八王之乱中引爆，两百年的南北乱世的大幕就此拉开。当权者和继承人之间的关系是很复杂，一个太过于强势的当权者，是很难培养出一个合格的继承人，因为当权人对权力的渴望是无穷的，所以当权人对过于精明强干的继承人是非常防备的。那也就是为什么历史上的一代明君最后培养出来的都庸庸碌碌之徒，导致盛世不过是昙花一现。这也就是为什么慈禧太后那么精明强悍的政治强人，培养出来的却是庸懦无能的光绪皇帝，自己亲政也好，百日维新也好，一点没学会中兴之君的隐忍与卧薪尝胆，就算慈禧太后不出面干预，立宪派最后胡闹的成果还不如北洋军阀。清末的大乱局已经不是不适合唐昭宗式的皇帝出来瞎折腾了，需要的是一位像雍正皇帝那样的狠人皇帝出来收拾残局了，开一方猛药，但是想要根除乾隆皇帝以来的百年积弊，还不如来个光武中兴算了。下面就是聊一聊中法战争了，中法战争中国最后还是赢了，但是带来的直接后果就是甲申易枢，以恭亲王为首的中枢被全面罢免，清廷的决策有点像秦昭襄王时期范雎主政期间破朔迷离的决策机制了，反正就是很迷，中央一帮清流和理学大师在吵吵洋务派干的违反祖宗法度，但是真遇到事又拿不出个对策，成事不足败事有余。</font>**

## <font face = STKaiti >中法战争</font>

## <font face = STKaiti >小站练兵</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;甲午战争充分说明了光绪帝的无能，这也就是为什么慈禧太后不愿意放权的根本原因，如果他是汉武帝那种雄才大略的人物，那么慈禧太后会舍得放权，帮助他成为一代雄主，不失为中国的彼得·大帝式人物。可惜光绪皇帝本身就是一个暗弱无能的人，他连唐昭宗都不算，再折腾也只会葬送大清已经不多的本钱，这一点从下面的百日维新可以看出来。甲午战争结束以后，清政府意识到旧式陆军，包括八旗、绿营和湘淮军已经不堪大用了，所以中枢决定在全国建立西式军队。但是财政一算账，三十六镇的新军根本就不是无法支撑，所以只能在小站这里先编练一镇新军看看效果如何。这里就要提到袁世凯的精明了，甲午战争开始之后清朝陆军在朝鲜是一泻千里，袁世凯也只能放弃他在朝鲜的太上皇地位灰溜溜地逃回京师，但是他逃回京师之后并没有放弃努力，毕竟他是袁保庆的嗣子和袁甲三的孙子，湘淮一脉还是都卖他一些面子的，也就是说他自己在军事上还有有一些市场的，他就做了一件事，写兵书并送给那些权贵获得了“知兵事”的美名。所以朝廷决定在小站编练新军时，袁世凯因为知兵的美称而被选为担当此任，不得不说，袁世凯这辈子转换门庭是被他玩明白了，投效翁同龢最终获得了这个机会。这里要插播一个冷知识，清朝有宰相之实只有四殿两阁一院大学士加上军机大臣，翁同龢是军机大臣但是一直被慈禧太后压着不让入阁。</font>**

## <font face = STKaiti >地方新军</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;袁世凯不得不说还是具有一定的军事才能的，小站练兵还是成果卓著的，辛亥革命的以后打得那些南方新军毫无还手之力。小站练兵从此就有了全新的含义，就是从明朝开始的亲兵这个制度被打破，打仗靠亲兵这种陋习被打破，这第一次从国家层面被打破。当然了，当曾国藩在湖南编练团勇的时候就已经开始打破这种陋习了，不过那种管理依靠宗族关系和师生情谊其实就是小站新军的骨架，但是湘军是自筹军费，这种模式能够维持到太平天国被消灭之后，之后曾国藩也不得不自解湘军表明忠心。袁世凯这种仿效湘军的模式的国家新军在中国历史上还是第一次，上下级之间全靠私人情谊维持的近代军队最终还是会消失，但是也算是近代化的关键一步了，这里还有一点，就是有了近代化的军队体系，但是缺乏配套的军事工业体系，这也制衡了未来五十年之内的中国军事力量发展，这就后话。小站练兵算是清末的一个标志性事件，也就意味着慈禧太后本人是倾向于革新，但是这种革新还是被怀疑的，所以保卫京师的还是传统军事军队。小站练兵清政府是下了血本的，一支七千人的部队一年的花费就要数百万两白银，这对于财政收入不过亿的清政府来说是一笔很大的开销了。还是时势造英雄，小站练兵造就了未来的北洋时代，北洋三杰也好，徐世昌也好，这些人构成了最初的民国政府的中枢，武夫当国也是小站练兵种下的苦果。小站练兵算是开了编练新军之先，当然了最早刘坤一、张之洞在湖北也编练了很多新军，但那都是地方财政支持的，随后清朝中枢就和那些地方督抚商量，中央出政策，地方出财政编练新军，但是全国三十六镇的目标现在看来还是不太可能。小站练兵是时代酿下的苦果，也是推动时代发展的动力，湘军系自十九世纪四十年代种下的因，最后在二十世纪成就了果。</font>**

## <font face = STKaiti >百日维新</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;很多人认为百日维新失败是因为袁世凯向慈太后告密导致的，其实百日维新的失败是注定的，中央枢臣没人买账，地方督抚也是应者寥寥，可以说是政令不出紫禁城。百日维新的声势是浩大的，但是声势浩大就有用吗？后来的学生运动规模更宏大，那不也是最后都是雷声大雨点小，闹着闹着就没有声音了。李敖在大陆演讲时有一句话是特别有道理的，他和国民党当局斗了二十年才获得了言论自由的权力。言下之意就是这些学生不要异想天开，自由和民主不是谁给的，而是自己争取到的。百日维新的序幕就是公车上书，但是公车上书其实没啥用，一群连功名都没有的人来说，闹一闹只不过哗众取宠罢了。不过康有为是个会造势的人，办了个强学会，中枢那些重臣都纷纷参与其中，连袁世凯送钱去了，可惜康有为这些保皇党是没有市场的，晚清的政局是由慈禧太后一个勉力支撑的，光绪皇帝其实就是个吉祥物，甲午战争结束以后中枢大臣和地方督抚已经达成了这个共识，所以光绪皇帝的折腾是无济于事的。甲午之后慈禧太后本人已经倾向于放权了，光绪皇帝抓住这个机会开始联合保皇党这派人开始搞事。光绪皇帝首先是上谕了一份“明定国是诏”，向天下宣称革故鼎新，可惜王朝末年，不改比改好。当然了，当年的百日维新的唯一直接成果就是京师大学堂，也就是北京大学的前身，也算是懦弱无能的光绪皇帝这一辈子干的一件大事了。百日维新，保皇党既没有控制军机处，也没取得地方势力的支持，就自己胡闹了一百多天，让人匪夷所思，我觉得东南那些地方实力派的态度也是最暧昧的，百日维新之后他们不支持慈禧太后废帝，百日维新的时候也不见他们支持光绪皇帝，他们只想维持目前的局面罢了。百日维新是轰轰烈烈，也是一剂猛药，但是天下吏治在道光皇帝时已经腐败不堪了，这些改革并不能及时地推进下去，就这和王安石的青苗变法一样，变法的主导者是好的，但是具体实施变法的人是小人，所以改革一定是要培养出新的既得利益集团，熬过二十年，使得既得利益集团能够主导政治舞台了。百日维新一顿胡闹，还是需要人出来收拾烂摊子，慈禧太后又从幕后走到了台前，开始重用保守派来进行反击，很快改革就要面临失败，保皇党们甚至幻想利用袁世凯来进行军事政变，但是之前已经提到了，清朝中枢还是相当清醒的，对小站新军并不放心，不仅严重管制其武器弹药等军事物资，还在京畿重地布置了大量的军事军队进行护卫，所以依靠袁世凯的军事政变是没有可能实现的。其实整件事在袁世凯告密之前已经泄密了，袁世凯也不会蠢到和保皇党合作去干这件不可能成功的事情，慈禧太后可谓是维护晚清政局的独木，真要这样八王之乱又会重现。</font>**

## <font face = STKaiti >戊戌政变</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;百日维新明眼人都知道是不可能成功的，光绪皇帝做此一搏属实让人疑惑，我本人是严重怀疑康有为的衣带诏是假的。慈禧太后又开始重新训政，光绪皇帝被囚禁瀛台，满朝上下都在猜测慈禧太后会立溥儁为新皇帝，但是以刘坤一为首的疆臣宣称“<span style="color: #F50F0B;">君臣之分久定，中外之口宜防</span>”，晚清重臣荣禄也不得不站出来表态，表示反对废立，中枢和地方势力都表示对废帝的反对，慈禧太后也不得不表示自己绝无废立之心，荣禄也算是清末中枢比较明事理的一个人，但是独木难支，很多时候也是身不由己。这些地方疆臣只想着平时和稀泥，能够维持大清国目前的局面就好了，和曾国藩一样，不想这大好天下重回乱世局面，但是这些努力最终都白费了，军阀乱战最终还是出现了，三十年的乱世为新朝浴火重生带来了希望。百日维新我认识是一场闹剧，光绪皇帝最要紧的事情是团结中枢重臣和地方实力派，在徐徐图之，改革不是请客吃饭，动嘴花点钱就行了，真是要真刀真枪地动别人的蛋糕，培养出忠于自己的既得利益集团，这点上光绪皇帝所受的帝王教育还是太浅薄了。</font>**

## <font face = STKaiti >庚子国变</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;百日维新虽然在中国历史上只激起了一些水花，但是其带来的副作用是贻害无穷的。中枢和地方实力派都是反对废立，慈禧太后内心是想废帝的，西方侵略者也实名支持光绪皇帝，这样慈禧太后和西方侵略者其实已经站到了对立面。义和团其实就是白莲教的残魂，慈禧太后肯定是不相信义和团的，她那个一个精明的老太太能够相信义和团那些鬼把戏是不可能的，只不过她觉得民心可用，朝堂之上也有很多满人亲贵和重臣支持这一行为。其实这件事就是中央失去了秩序，那些只懂得经学道义的大臣希望利用义和团打击西方侵略者，可惜连一个个小小的使馆区都不打下来，最终京师重地失去了重地，几十万的青壮汉子在京师胡作非为，清朝中枢已经失去了约束他们的能力，最终导致了庚子国变，八国联军从大沽口登陆一路打到北京，慈禧太后连夜出逃西安。清朝中枢这个时候已经威权不再了，慈禧太后一路出奔西安，连吃的都没有，还是岑春暄一个人护送慈禧太后一行人到西安。但此时的大清国已经有了亡国之兆，东南督臣竟然联合起来弄了一个东南互保，简直是不可想象，差不多等于东南独立了。这些督抚还是讲情面，只是说慈禧太后向十三国宣战诏书是乱命，不敢受也，还给慈禧太后留了最后一点情面。这个时候有人建立李鸿章建立中华帝国，由李鸿章出任第一任总统，但是后周郭威的先例在先，已经是半只脚入土的人没必要去过个总统的瘾。李鸿章就北上和甲午时一样收拾残局，和庆亲王一起和谈解决目前的问题，此次和谈赔了四万万两白银还是能接受的，但是让列强在中国腹地驻扎重兵是万万不能接受的，中国的主权基本从此时已经丧失殆尽了，国家威权已经难以压制这些地方实力派。如果说甲午的时候，中国人在思考自己的体制问题，认识体制内的革新能够解决目前的问题，那么庚子以后有识之士思考的就是体制是不是出现了问题，需要对目前陈旧的体制进行革新，中国革命的意识是在一次又一次的丧师失地之后觉醒的，中国这头雄狮终于意识了全面向西方学习的重要性。这里清末有三屠都出场了，官屠岑春煊，民屠袁世凯，钱屠张之洞都正式登上了政治舞台。庚子国变正式敲响了亡国钟，慈禧太后也意识到问题的紧迫性，开始派出五大臣出洋考察宪政，开始积极接触外国使节，缓缓推进百日维新的那些革新措施，但还是积弊难返，王朝末年想要全面改革是不现实的，除非来一次王莽乱汉那种事情解放社会生产力。</font>**

## <font face = STKaiti >变法与维新</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;庚子国变之后，慈禧太后和那帮有识之士都知道大清朝腐朽的制度已经是不改不行了，地方实力派开始动了分权的想法，中枢里像荣禄那样的能人也没有了，继任的庆亲王虽然忠心但是能力不太行，连同治中兴那样的帝国余晖都难以重现了。慈禧太后也知道自己日薄西山了，就开始考虑身后的事情，积极推动变法，任用汉人，借阅兵的机会剥夺了汉臣袁世凯的军权，调入中枢辅助自己推行改革。这里要说一下，袁世凯虽然后世被誉为窃国大盗。但是还是颇有英雄之风，广散家财收揽天下英雄之心。这也就是为什么辛亥革命的时候北洋六镇基本都听袁世凯的话，因为袁世凯舍得放权花钱，这样的领导谁不想拼死效命。慈禧太后在庚子国变之后就开始布置自己的身后事了，将袁世凯调入中枢就是削弱天下汉臣的实力，怕自己死后压不住，但是东南互保已经撕破了清朝政府最后的脸面，大有高声一呼，天下景从的局面。到这个时候，改革已经是箭在弦上，不得不发了，她亲自主持了第三次改革，其深度和力都不是百日维新这样的小打小闹能够比得了的，这次改革是深刻的，但是在京畿之地之外的地方成果寥寥，因为地方实力派已经在自己主导地方上的改革了，1901年废科举是一个影响深远的事情。废科举看似是一件好事情，但是其从本质上伤害了士绅阶级的根本利益，造成了天下有识之士的人心离崩，大批有钱人家子弟都东渡东洋留学，这个时候革命浪潮已经扎根在下层统治阶层的心里，开始腐蚀清廷的统治基础。1905年慈禧太后派出五大臣出洋考察宪政，我想五大臣在考察了日本和欧洲各国之后，心里有了决定，一定要回国推进立宪进程。当时满洲亲贵还是有不少能人的，像端方、铁良和良弼还都是能干的，慈禧太后在的时候也用这些人来干宪政，慈禧太后的手段还是很老辣的，知人善用，还能压住那些地方实力派。中国君主立宪就不得不提到杨度这个人，他是当时帝王术大成者王恺运的一生，就是他在背后给袁世凯出主意，此人一生信仰先是研究君主立宪，后怂恿袁世凯复辟，后又加入国民党，最后加入共产党，一生可谓是经历了中国近代史的波澜壮阔，是近代中国历史的见证人，我想他最近是看到了共产党的言行一致，深感自己折腾了一生也没有为黎民苍生挣得温饱，最后决定参加中国共产党。1906年，这个时间节点，自知自己时日无多的慈禧太后决定正式开始立宪，颁布了九年立宪计划，主要就是确定君主</font>**

## <font face = STKaiti >九年立宪</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;立宪政体，确保皇权的万世一系，主要措施就是地方自治和开国会，谘议局、资政院和地方自治，这些是立宪当中的实质内容，这也代表了清廷中枢决定正式与地方实力派合作，让渡出部分权力。潘多拉的魔盒要被打开了，地方实力派被慈禧太后压着的时候还不敢跳到明面上要求分权，但是慈禧太后一死，地方实力派就差直接造反了。这里说一句公道话，慈禧太后庚子国变之后做的还算不错，也是有声有色，废科举，推行立宪，做的比光绪皇帝好得多。下面就到了1908年，这一年清廷中枢权力出现了短暂的真空了，我个人是倾向于慈禧太后死之前把光绪皇帝毒死了，光绪皇帝头天驾崩，慈禧太后第二天也驾鹤西去，这让人不得不怀疑，慈禧太后把光绪皇帝毒死也很正常，自己死了，光绪皇帝出来瞎闹，那就山河永无宁日了，不如一波直接带走来得省心。光绪皇帝的弟弟醇亲王的儿子溥仪被慈禧太后选中登上帝位，醇亲王被封为摄政王，和隆裕皇太后一起决定国事，这个辛酉政变一样，姑嫂一起把持朝政，但是醇亲王不是恭亲王，隆裕皇太后也不是慈禧太后，这两个人的历练和手段都不值得一提。慈禧太后还特别提到，她死后再也不允许女人干政。慈禧太后的死，清朝就差最后的氧气管没拔了，这里醇亲王太年轻了，没有被历练到位，并不是一个合格的政治家。大清朝的命运从此注定，以摄政王始，以摄政王终。</font>**

## <font face = STKaiti >暗流涌动</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;醇亲王开始了自己的摄政生涯，那个曾南下收权的铁良被排挤出了中枢决策权，这个人也是满人亲贵算是能干的，这些能干的人最后也都在清帝退位前隐退了，唉，亡国之前忧心国事的人都不得不急流勇退。1908年以后，那些年轻的满人亲贵又开始嚷嚷着收权了，把当时的地方督臣之首张之洞掉到中枢担任军机大臣，和袁世凯一起赞襄朝务，算是架空了他们手中的权力。这些人以为自太平天国之后中央就丧失的权力他们轻轻松松就能收回来了，得之易者失之易也，这个简单的道理他们都不懂，就由着自己的性子胡来。这段时间还有一个争议，就是要不要杀袁世凯，让摄政王报自己哥哥光绪皇帝的仇，但是在张之洞的劝说下，隆裕皇太后还是最终决定以足疾开缺袁世凯回原籍修养，袁世凯只能灰溜溜地回到项城归隐，但是自己还是远程遥控北洋六镇。一开始的时候袁世凯还是担心自己被杀，但是随着时间的增加，袁世凯的老部下也都纷纷来访，让袁世凯放下了心，静待事变重新出山收拾烂摊子。这里将视线再次回到清廷中枢，这一段时间中枢多事，那些地方实力派纷纷上京要求早日开国会，实现地方自治，他们并不关心君主立宪是什么样的政体，他们只关心自己的手中的权力如何扩大，他们给清朝续命不是为了天下黎民，而是士绅集团的自身利益罢了。各地谘议局和资政院的议员纷至沓来，到中央要求速开国会，缩短九年立宪的时间，年轻的醇亲王没有办法，只能一次次应了他们要求缩短立宪时间，最后缩短到五年，1913年正式召开国会，没想到一次次的妥协最后亲手埋葬了大清朝，果然妥协只会害了自己。大厦将倾，不是妥协能够解决的，必须有慈禧太后和恭亲王这样有大智慧的人才能解决问题，已经交给别人的东西，不是那么轻易地能够要回来的，拿回来必然是十倍或者百倍的代价。中枢汉臣张之洞死后，中央和地方已经失去了平衡，那就意味着地方和中央已经是表面和睦了。</font>**

## <font face = STKaiti >新军与辛亥革命</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;这里就要谈一谈南方诸省的新军了，这些新军大多是喝过洋墨水的，心向革命，革命党已经把新军中下层完全腐蚀了，这也就为什么革命党屡屡失败的原因，只能获得有限的下层支持，缺乏领导层的助力。这里又要提到一个人，盛宣怀，当时是大清邮传部尚书，就是由他提出调动湖北新军平定四川保路运动。此时教科书赫赫有名的皇族内阁也已经成立，有识之士和地方实力派已经完全失望了，他们更多可惜和倾向革命了。这里四川保路运动是指盛宣怀为了填补财政窟窿，将已归商办的粤汉、川汉铁路收归“国有”，任命端方为督办粤汉川汉铁路大臣，并与英、法、德、美四国银行团签订《湖广铁路借款合同》。南方各省的反应最为激烈，以四川斗争尤为激烈，立宪派绅商首先发起保路，持有租股的农民争相入会，参加者达数十万人。川督赵尔丰带领湖北新军镇压保路运动，这里其实清廷已经把支持立宪的地方实力派逼到了政府的反对面，中央和地方已经彻底离心离德，连表面的和睦也已经不要了，实在让人费解。湖北新军主力被调走了，革命党以为大事可成，就准备剩下的新军革命，但是在租界内制造炸药导致事情泄露，导致革命党名单被湖广总督瑞澄获得，他要是当机立断给烧了也行，按照名单抓人也行，他一样都没选择，选择观望，最后导致了武昌事变。清廷对这些新军其实是很不信任的，子弹等军资要物都被严格管控，其实这场起义是个偶然，只不过底层官兵的自发行为，武昌起义的第一枪不重要，但是湖广总督瑞澄第一时间跑了，导致兵败如山倒，新军拿到了紧需的子弹等物资，辛亥革命就这样开始了，所有人都没想到。经过一夜的激战，新军控制了武昌城，但是此时新军群龙无首，只得推举黎元洪为首，这样一个旧式官僚就这样革命了，让所有人大跌眼镜，这样式的革命也戏剧般地在汉地十八省一次次重演。保路运动彻底寒了士绅集团的心，湖北谘议局议长汤化龙、副议长张囯溶主动和革命党合作，汤化龙的胞弟汤芗铭是大海军参谋，颇有天下谁人不革命的感觉？这里已经说明了地方实力派与中央彻底分道扬镳了，武昌首义，天下景从，当时清廷还是颇有实力的，派出荫昌进攻武昌，但是北洋六镇不听朝廷命令，只有袁世凯能够调得动。</font>**

## <font face = STKaiti >清帝退位</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;无奈之下，摄政王只能放弃权力隐退，袁世凯作为全权大臣掌握朝政，其实革命军的战斗力在北洋军面前就是一个渣渣，革命军甚至连张勋那样的巡防营打起来都费劲，但是南京反正对清廷是一个沉重的打击，南京临时政府有了法理上的正统性。北洋军只是稍微打了一下，就把武昌城攻下了，但是此时南京被光复，除非北洋军能够荡平汉地十八省，才能平定辛亥革命。袁世凯通过自己的算计，决定南北和谈，北方代表伍廷芳和南方代表唐绍仪最终达成一致，让清帝退位，实行民主政体。这里有个小插曲，此时北方的革命人士也在加紧行动，满人最后一个能人良弼被人刺杀，满人中再也没人能够反对清帝退位了。袁世凯是辛亥革命最大的功臣，当时的北洋军是中国最强大的军事力量，这在未来的十几年也是，就这样，在之前的中枢汉臣和地方实力派的合作下，清政府成为了历史，袁世凯也出任了中华民国第一任大总统，就这样，大清亡了，但是图强救亡的脚步并不能停下来，我们还需要继续努力。</font>**
<center><span style="font-family: STKaiti; font-size: 40px; color: #FC0101; font-weight: bold">大清国皇帝退位诏书</span></center>
<span style="font-family: STKaiti; font-size: 25px; color: #FC0101; font-weight: bold">&emsp;&emsp;朕钦奉隆裕太后懿旨：前因民军起事，各省响应，九夏沸腾，生灵涂炭，特命袁世凯遣员与民军代表讨论大局，议开国会，公决政体。<br>
&emsp;&emsp;两月以来，尚无确当办法，南北睽隔，彼此相指，商辍于途，士露于野，徒以国体一日不决，故民生一日不安。<br>
&emsp;&emsp;今全国人民心理多倾向共和，南中各省既倡议于前；北方诸将亦主张于后，人心所向，天命可知，予亦何忍因一姓之尊荣，拂兆民之好恶。用是外观大势，内审舆情，特率皇帝将统治权公之全国，定为共和立宪国体。<br>
&emsp;&emsp;近慰海内厌乱望治之心，远协古圣天下为公之义。袁世凯前经资政院选举为总理大臣，当兹新旧代谢之际，宣布南北统一之方，即由袁世凯以全权组织共和政府，与民军协商统一办法。总期人民安堵，海宇又安，仍合汉满蒙回藏五族完全领土为一大中华民国，予与皇帝得以退处宽闲，优游岁月，长受国民之优礼，亲见郅治之告成，岂不懿欤！</span>

## <font face = STKaiti >一脉相承，有始有终</font>

**<font face = STKaiti color = Black size = 4>&emsp;&emsp;当年王闿运劝曾国藩反了清朝，曾国藩用手蘸茶水在桌上写“妄”字，表明了自己的心迹，但是也同样认为清祚不长，没想到湘军系的传人袁世凯最后亲手将大清朝变成了历史。《左传》有云，君以此兴，必以此亡，湘军给大清朝续命，湘军最后又成了大清朝的掘墓人，历史的滚滚车轮不是一族一姓能够阻挡的。</font>**
**<font face = STKaiti color = Black size = 4>&emsp;&emsp;晚清史就是中央和地方之间的博弈，中央缺少了一个精明强悍的能人，那么地方实力派就会去争取自己的权力。自太平天国始，地方自治已经是大势所趋了，只有慈禧太后能够凭借个人威信和手段能够延缓这一趋势，但是年轻的满人亲贵少不遇事，最终让湘淮系的袁世凯革故鼎新，当了一次刘裕。</font>**
